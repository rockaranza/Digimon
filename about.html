Acerca de mi proyecto

Proyecto realizado para finalizar el Módulo 2 del Bootcamp Frontend de Ecamp. Utilizando JS se debe consultar la API de Digimon y listar los elementos que trae. Ademas implementar un buscador para mostrar los datos de un Digimon. Se pidió agregar otro evento en JS. Lo implemente con un botón que hace una busqueda aleatoria. Implemente un algoritmo que me devolviera un numero al azar entre el rango de elementos que trae el array de la API, y con este numero buscar el elemento que deseo mostar.

Recursos utilizados
  • API DIGIMON
  • BootStrap
